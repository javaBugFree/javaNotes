
在*CategoryController*中处理请求：
```java
// http://localhost:8090/categories/list-by-parent  
@GetMapping("/list-by-parentId")  
@ApiOperation("根据父级类别查询子级类别列表")  
@ApiOperationSupport(order = 420)  
@ApiImplicitParam(name = "parentId", value = "父级类别ID", required = true, example = "9527", dataType = "long")  
public JsonResult<List<CategoryListItemVO>> listByParentId(Long parentId) {  
    log.debug("开始处理【根据父级类别查询子级类别列表】的请求，参数：{}", parentId);  
    List<CategoryListItemVO> list = categoryService.listByParentId(parentId);  
  
    return JsonResult.success(list);  
}
```
